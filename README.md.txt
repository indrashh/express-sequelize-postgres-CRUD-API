Express Sequelize Todo App  
Aplikasi Todo sederhana menggunakan Express.js dan Sequelize dengan PostgreSQL.  

Fitur  
- CRUD Todo  
- Menggunakan Sequelize ORM  
- Database PostgreSQL  
 
Teknologi yang Digunakan
- Node.js
- Express.js
- Sequelize (ORM)
- PostgreSQL
- DBeaver (untuk mengelola database)
- Postman (untuk testing API)
